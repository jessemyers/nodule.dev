# nodule

`nodule` is a collection of open-source Node libraries for building opinionated API gateways.

 - [Opinionated configuration for Node applications](https://github.com/globality-corp/nodule-config)
 - [Opinionated logging for Node projects](https://github.com/globality-corp/nodule-logging)
 - [Opinionated OpenAPI (Swagger) Client](https://github.com/globality-corp/nodule-openapi)
 - [Node Express Conventions](https://github.com/globality-corp/nodule-express)
 - [Node GraphQL Conventions](https://github.com/globality-corp/nodule-graphql)
 - [Opinionated Memcached Client](https://github.com/globality-corp/nodule-memcached)
